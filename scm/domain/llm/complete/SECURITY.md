# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✅        |

## Reporting a Vulnerability

Report security issues privately to the SWE Engineering Labs security team at
`security@swelabs.io`. Do not open public issues for security-sensitive reports.

We aim to acknowledge reports within 3 business days and to provide a remediation
timeline within 10 business days.
