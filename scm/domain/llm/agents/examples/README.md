# Examples

This directory contains example code demonstrating the edge-domain-agent primitives.

## Note

Agent implementations are provided by plugins (e.g., `edge-plugin-llmboot`).

Example code for using agents is typically found in the plugin crates, not here.

For examples of how to:
- Load agents from YAML specs
- Execute skills
- Discover agents and their capabilities

See the [edge-plugin-llmboot](../../edge/plugins/llmboot) plugin documentation.
